# An-lisis-del-Mercado-Energ-tico-Colombiano-y-Predicci-n-de-Precios
El proyecto fue el desarrollo de una aplicación Python que analiza y predice el Precio Spot de la energía en Colombia. Se integraron múltiples APIs para confirmar que la vulnerabilidad hídrica es el principal factor de riesgo, culminando con la creación de un modelo predictivo base.
